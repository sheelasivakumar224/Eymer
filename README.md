# Eymer
Timer based on 20-20-20 Rule

<img width="1911" height="1063" alt="Screenshot 2026-06-12 190547" src="https://github.com/user-attachments/assets/719d1825-1a39-411c-8846-2cbe4ef635a9" />

1. Press <br>
```Win + R```
2. Type <br>
```shell:startup```

3. Create a shortcut for eymer.exe and place it in the Startup folder that opens. Done! Now you can monitor your screen time and the breaks. 
4. To confirm it is running, check the system tray/notification area (bottom-right corner of the taskbar). You should see the Eymer logo there.
